### 动画类描述 - utility_class文件

#### `Timer_animation` 类
`Timer_animation` 类用于执行动画效果。

- **功能**:
  - 启动和停止动画
  - 控制动画的持续时间和间隔
  - 处理动画的重置和还原状态

#### `Timer_animationgroup` 类
`Timer_animationgroup` 类用于管理多个动画的执行顺序。它允许开发者将多个动画组合在一起，并定义它们的执行顺序，确保动画之间的协调。

- **功能**:
  - 添加多个动画到动画组
  - 定义动画的执行顺序（顺序、并行）
  - 支持动画完成后的回调函数
